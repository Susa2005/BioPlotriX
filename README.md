# BioPlotriX
**An R Shiny-based Biochemical Data Visualization & Standard Curve Automation Tool**

BioPlotriX is an interactive web-based biochemical graph plotting tool built using **R Shiny**.  
It automates analysis from OD (Optical Density) values, generates standard curves, computes unknown concentrations, and produces publication-ready plots for biochemical experiments.

---

## Features

- Upload OD (optical density) values
- Generate standard curves dynamically
- Polynomial/linear model curve fitting
- Automatic computation of unknown concentrations
- Interactive visualization of biochemical assay data
- Publication-ready graphs (PNG/Download options)
- Supports multiple biochemical methods:
  - Lowry‚Äôs Method (Protein Estimation)
  - Anthrone Method (Carbohydrate Estimation)

  - And additional quantitative assays
